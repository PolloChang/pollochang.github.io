設定 Linux 檔案瀏覽器 Files 預設的 Columns

2024-07-01 linux linux GNOME

  1. 使用以下命令來查看當前設置：
gsettings get org.gnome.nautilus.list-view default-visible-columns
  1. 設置你想要的欄位，例如：
gsettings set org.gnome.nautilus.list-view default-visible-columns "['name', 'size', 'type', 'date_modified']"

default-visible-columns values

  1. name: 檔案或目錄的名稱
  2. size: 檔案的大小
  3. type: 檔案的類型
  4. date_modified: 檔案的修改日期
  5. date_accessed: 檔案的訪問日期
  6. owner: 檔案的所有者
  7. group: 檔案的群組
  8. permissions: 檔案的權限
  9. detailed_type: 檔案的詳細類型（例如“MPEG-4 影片”、“PNG 圖片”）
