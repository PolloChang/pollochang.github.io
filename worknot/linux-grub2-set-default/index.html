Linux Grub2 Set Default(手動設定開機時要啟動的 linux 核心)

2024-07-03 Linux CLI GRUB kernel

手動設定開機時要啟動的 linux 核心

操作步驟

  1. 查看目前已經有安哪些核心
  2. 選擇核心
  3. 重新生成GRUB配置
  4. 重新開機
  5. 檢查
# 1. 查看目前已經有安哪些核心
grep ^menuentry /boot/grub2/grub.cfg

# 2. 選擇核心
grub2-set-default 0

# 3. 重新生成GRUB配置
grub2-mkconfig -o /boot/grub2/grub.cfg

# 4. 重新開機
reboot

# 5. 檢查
uname -a
