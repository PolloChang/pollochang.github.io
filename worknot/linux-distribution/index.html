Linux 發行版

Debian

  • https://www.debian.org/

Fedora

特別發行版

  • Skarnet 的軟體堆疊可以被認為是一個發行版（肯定透過 lh-bootstrap） - 而且會是一個有趣的競爭者

  • Oasis - 靜態連結的發行版，不怕用危害較小（但通常較不好用）的替代方案來取代常見的軟體

  • KISS Linux - 非常簡單的發行版，相當模組化且可駭。套件管理器是一個 shell script

  • Carbs Linux - KISS 的分支，有一個更像 xbps 的套件管理器（仍然用 sh 寫成）

  • Venom Linux - 比上述選項更像典型的發行版，但仍然相當簡潔。也有一個基於 sh 的套件管理器
