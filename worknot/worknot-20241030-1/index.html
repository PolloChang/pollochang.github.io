硬體設備-hba

2024-10-30 工作雜記 linux multipath

最近工作中遇到需要設定光纖設備，這邊我稍微紀錄一下過往常用的指令

查詢 hba 型號: ``

[root@test-host ~]# lspci -nn |egrep -i "fibre|hba"
17:00.0 Fibre Channel [0c04]: QLogic Corp. ISP2532-based 8Gb Fibre Channel to PCI Express HBA [1077:2532] (rev 02)
17:00.1 Fibre Channel [0c04]: QLogic Corp. ISP2532-based 8Gb Fibre Channel to PCI Express HBA [1077:2532] (rev 02)

重新掃描 scsi: echo "- - -" > /sys/class/scsi_host/<host>/scan

for host in ls /sys/class/scsi_host/;do echo "- - -" >/sys/class/scsi_host/${host}/scan; done

查詢 FC WWN: ``

[root@test-host ~]# cat /sys/class/fc_host/host11/port_name
0x21000024fg400000
[root@test-host ~]# cat /sys/class/fc_host/host12/port_name
0x21000024ff400000
