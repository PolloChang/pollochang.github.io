Redmine 資料庫架構與 Table / Column 分析報告
#

註記

本文件針對連線環境 nb-vm114-redmine 進行完整架構解析與欄位彙整。分析時間：2026-06-28。新增全模組 Mermaid ERD 關聯圖。

📊 架構概述 (Executive Summary)
#

  • 資料表總數 (Total Tables): 58 個
  • 欄位總數 (Total Columns): 440 個
  • 主要模組領域: 專案成員、使用者權限、議題追蹤、自訂欄位、Wiki文件、討論區、版本庫整合、系統設定。
