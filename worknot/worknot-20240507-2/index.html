網頁連線問題

2024-05-07 工作雜記

問題

You cannot visit domain.example.com right now because the website uses HSTS.

說明

該網站啟用了 HSTS（HTTP Strict Transport Security）功能，這意味著您的瀏覽器記住了您僅能使用安全的 HTTPS 連接方式訪問該網站，以增強安全性。這可以防止您的瀏覽器通過不安全的 HTTP 連接訪問該網站。

可能的狀況

  1. 憑證過期
❯ curl -v -k https://volunteer.ntpc.gov.tw 2>&1 | grep -E "Connected to|subject|expire"
* Connected to volunteer.ntpc.gov.tw (61.60.98.240) port 443 (#0)
*  subject: C=TW; L=\Uxxxx\Uxxxx\Uxxxx; O=\Uxxxx\Uxxxx\Uxxxx; OU=\Uxxxx\Uxxxx\Uxxxx; OU=\Uxxxx\Uxxxx\Uxxxx\Uxxxx; CN=domain.example.com; serialNumber=0000000010045160
*  expire date: Aug  8 00:00:00 2020 GMT
