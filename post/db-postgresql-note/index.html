安裝時設定項目

安裝前檢查項目

  • 作業系統版本
  • 資料庫空間需要獨立空間
  • 資料庫備份需要獨立空間
  • 稽核確認: 資料庫連線稽核
  • 稽核確認: 資料庫DDL異動紀錄
  • 定期備份排程

資料庫備份

  • 定期備份排程

user: postgres

0 1 * * * pg_dump -Fc -c dbName > /data/backup/dbName.$(date +%Y%m%d%H%M%S).dump -O # 備份
0 6 * * * /usr/bin/find /data/backup/* -type f -mtime +7 -exec rm -f {} \; # 刪除備份

變更權限

  • change table owner
ALTER TABLE [tableName] OWNER TO [userName];.
  • change schema owner
ALTER SCHEMA public OWNER TO [userName];
